# 🎶Reop Client
## 😊简介：
本项目是基于Reop HTML 项目的单独前端项目，旨在替代Reop HTML前端
## 😉概述：
 - 本项目是`为了ReopHTMl项目而开发`的。在开发ReopHTML项目时,本人曾立下flag说要使用vue重构项目
 - 本项目使用`vue 3`。
 - 本项目使用`mysql 8.0`作为主数据库。
 - Node.js 版本选择 `v22.5.1` 稳定版。
 - `nvm`做为nodejs的版本管理具。
 - `npm v10.8.2`作为本项目的包管理器
